# Running and Debugging Tests

You can use `--ui` flag while running tests to open Trace Viewer for [Debugging the tests](https://playwright.dev/docs/test-ui-mode).

```
npx playwright test --ui
```

You can do the following:

- Easily walk through each steps of tets
- Visually see what was happing before test, during test and after each step of test
- Locator picker
- And much more...
